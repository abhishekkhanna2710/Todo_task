<br />
## If you have to run in your system(Locally)

You should have to clone this repo

```bash
https://github.com/abhishekkhanna2710/React_CRUD.git
```

Now open the terminal

Install dependencies

```bash
  npm install
```

Start the Client

```bash
  npm run dev
```

Start the Server

```bash
  npm start
```
